## **Вывод чисел в диапазоне**

```python
num = 1
while num <= 5:
    print(num)
    num += 1
```

Этот пример используется для объяснения базовой структуры цикла `while` и демонстрации того, как программа может выполняться, пока условие истинно.


## **Таблица умножения**

```python
num = 1
while num <= 10:
    print(f"5 * {num} = {5 * num}")
    num += 1

```

Задача по выводу таблицы умножения использует цикл `while` для повторения блока кода и вывода результатов.


## **Сумма чис****ел**

```python
total = 0
num = 1
while num <= 10:
    total += num
    num += 1
print(f"Сумма чисел от 1 до 10: {total}")

```

Пример демонстрирует использование цикла для нахождения суммы чисел в диапазоне.


## **Чтение данных из файла**

```python
with open("data.txt", "r") as file:
    line = file.readline()
    while line:
        print(line)
        line = file.readline()

```

Цикл используется для построчного чтения данных из файла и их обработки.


## **Повторение за****просов пользователя**

```python
user_input = input("Введите пароль: ")
while user_input != "secret":
    print("Неверный пароль. Попробуйте еще раз.")
    user_input = input("Введите пароль: ")
print("Доступ разрешен.")

```

В этом примере цикл `while` применяется для запроса пароля от пользователя до тех пор, пока не будет введен верный пароль.


## Ожидание события

```python
is_event_occurred = False
while not is_event_occurred:
    # Ждем события или условия
    if check_for_event():
        is_event_occurred = True

```

Цикл может использоваться для ожидания событий, например, завершения процесса или получения данных.
